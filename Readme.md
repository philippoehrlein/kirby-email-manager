# Kirby Email Manager Plugin

The Kirby Email Manager Plugin is a powerful tool for developers, designed to simplify email form creation in Kirby CMS projects.

## Features

- Easy email template selection
- Customizable form fields
- Multi-language support
- GDPR compliance options
- Flexible email routing based on form subject
- Robust form validation

## Installation

### Manual Installation

1. Download the plugin
2. Unzip the file and place the `kirby-email-manager` folder in your `site/plugins` directory
3. Rename the folder to `kirby-email-manager`
