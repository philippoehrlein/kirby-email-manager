# Booking Request Form

A request form for booking services or accommodations. Includes fields for contact details, desired booking period, and number of participants.

## Installation

Copy the files to your Kirby site:

- `blueprints/emails/booking-request-form.yml` → `site/blueprints/emails/`
- `templates/emails/booking-request-form/*` → `site/templates/emails/booking-request-form/*`

## Setup

1. Create a new page in the Panel
2. Add the Email Manager tab or block to your page blueprint
3. Select the form as the template
4. Configure recipient email and success message
5. Done!
