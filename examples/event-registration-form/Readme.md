# Event Registration Form

A form for registering participants for events. Includes fields for personal details, event date, participant count, and an optional newsletter signup.

## Installation

Copy the files to your Kirby site:

- `blueprints/emails/event-registration-form.yml` → `site/blueprints/emails/`
- `templates/emails/event-registration-form/*` → `site/templates/emails/event-registration-form/*`

## Setup

1. Create a new page in the Panel
2. Add the Email Manager tab or block to your page blueprint
3. Select the form as the template
4. Configure recipient email and success message
5. Done!
