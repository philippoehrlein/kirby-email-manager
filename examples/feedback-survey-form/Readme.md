# Feedback Survey Form

A form for collecting feedback from users. Offers rating options and a text field for detailed comments.

## Installation

Copy the files to your Kirby site:

- `blueprints/emails/feedback-survey-form.yml` → `site/blueprints/emails/`
- `templates/emails/feedback-survey-form/*` → `site/templates/emails/feedback-survey-form/*`

## Setup

1. Create a new page in the Panel
2. Add the Email Manager tab or block to your page blueprint
3. Select the form as the template
4. Configure recipient email and success message
5. Done!
