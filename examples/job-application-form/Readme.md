# Job Application Form

An application form designed for job seekers. It allows candidates to provide contact details, upload their resume, and add a motivational statement.

## Installation

Copy the files to your Kirby site:

- `blueprints/emails/job-application-form.yml` → `site/blueprints/emails/`
- `templates/emails/job-application-form/*` → `site/templates/emails/job-application-form/*`

## Setup

1. Create a new page in the Panel
2. Add the Email Manager tab or block to your page blueprint
3. Select the form as the template
4. Configure recipient email and success message
5. Done!
