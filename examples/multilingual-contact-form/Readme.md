# Multilingual Contact Form

A basic contact form with name, email, subject and message fields in German and English.

## Installation

Copy the files to your Kirby site:

- `blueprints/emails/multilingual-contact-form.yml` → `site/blueprints/emails/`
- `templates/emails/multilingual-contact-form/*` → `site/templates/emails/multilingual-contact-form/*`

## Setup

1. Create a new page in the Panel
2. Add the Email Manager tab or block to your page blueprint
3. Select the form as the template
4. Configure recipient email and success message
5. Done!
