# Simple Contact Form

A basic contact form with name, email, and message fields.

## Installation

Copy the files to your Kirby site:

- `blueprints/emails/simple-contact-form.yml` → `site/blueprints/emails/`
- `templates/emails/simple-contact-form/*` → `site/templates/emails/simple-contact-form/*`

## Setup

1. Create a new page in the Panel
2. Add the Email Manager tab or block to your page blueprint
3. Select "Simple Contact Form" as the template
4. Configure recipient email and success message
5. Done!
