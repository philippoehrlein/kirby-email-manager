# Support Request Form

A form for handling customer support inquiries. Users can select an issue category, describe their problem, and optionally upload a file.

## Installation

Copy the files to your Kirby site:

- `blueprints/emails/support-request-form.yml` → `site/blueprints/emails/`
- `templates/emails/support-request-form/*` → `site/templates/emails/support-request-form/*`

## Setup

1. Create a new page in the Panel
2. Add the Email Manager tab or block to your page blueprint
3. Select the form as the template
4. Configure recipient email and success message
5. Done!
